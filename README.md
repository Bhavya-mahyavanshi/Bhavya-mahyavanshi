### 🎯 HireTrack
**Canadian Job Application Tracker**

Full-stack platform that auto-scrapes job postings, calculates skill match scores, and sends follow-up reminders.

**Stack:** Java · Spring Boot · MySQL · JWT · Jsoup · Next.js · React

**Highlights:**
- REST API with Spring Security + JWT auth
- Jsoup web scraping — paste URL, get job data
- Skill gap algorithm — match score 0–100%
- Spring `@Scheduled` email reminders
- CI/CD via GitHub Actions · Deployed on Render + Vercel
